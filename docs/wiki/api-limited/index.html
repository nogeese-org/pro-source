The Nogeese Pro API (Limited) allows users to communicate with Nogeese Pro. The API has it's own set of commands for developers: - oneAuth() - Authenticate a Nogeese One server account. - proConnect("test") - Connect to a cloud (Acording to Nogeese Pro's ToS you shouldn't modify the things on the server, but this is the source so have fun!) - proSetVar(5, "Variable") - Set a variable - proGetVar("Variable") - Get a variable The Projects used in this API: - Nogeese One (Source) - ngone.js (Source) - ngpro.js (Source) - ngonecloud (Source)
